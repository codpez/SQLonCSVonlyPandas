SQLonCSV - Cómo usar

Propósito

SQLonCSV permite ejecutar queries SQL sobre archivos CSV cargados desde tu navegador.

Queries Soportadas

  • SELECT - Soporta WHERE, ORDER BY y LIMIT.
  • INSERT - Agregar nuevas filas a tus datos.
  • UPDATE - Actualizar filas existentes basadas en condiciones.
  • DELETE - Eliminar filas específicas usando condiciones.

Ejemplos de Uso

  • SELECT id, name FROM employees WHERE age > 30;
  • INSERT INTO employees (id, name, age) VALUES (27, 'Grace', 26);
  • UPDATE employees SET salary = 60000 WHERE id = 3;
  • DELETE FROM employees WHERE department = 'IT';

Mensajes de Error Comunes

  • "El archivo debe ser un CSV": Asegúrate de subir un archivo con extensión .csv.
  • "Error en SELECT": Revisa la sintaxis de tu query.
